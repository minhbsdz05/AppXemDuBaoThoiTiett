# 🌦️ Weather Forecast App (Flutter)

Ứng dụng dự báo thời tiết thông minh được phát triển bằng Flutter, cung cấp:
- Thời tiết thời gian thực
- Dự báo nhiều ngày
- Phân tích bằng AI

---

## 🚀 Features

- 🌍 Xem thời tiết theo vị trí hiện tại  
- 🔍 Tìm kiếm thành phố  
- 📅 Dự báo theo ngày & giờ  
- 🗺️ Bản đồ thời tiết  
- 🤖 AI phân tích & tóm tắt  
- ⚠️ Cảnh báo thời tiết (mưa, bão, sương mù...)  
- 📊 Phân tích dữ liệu  
- ❤️ Quản lý địa điểm yêu thích  
- ⚙️ Tùy chỉnh (theme, °C/°F)  

---

## 🧠 Technologies Used

- Flutter (Dart)  
- REST API (Weather API)  
- Provider (State Management)  
- Clean Architecture  
- JSON & HTTP  
- AI Summary
